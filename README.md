# Card
マルチプレイヤーカードゲーム
ここには開発者向けのメモを記載する

## 開発の進め方
1. `/feature` を実行して新機能を実装する（ヒアリング→実装→テストまで自動で進む）
2. PlayMode: Window → General → Test Runner → PlayMode タブ → Run All で自動テストを実行する
3. Unity Editor で Play して動作確認する
4. 問題なければ `/ship` を実行してコミット・ドキュメント更新まで行う

## 機能一覧
- BGMとSEの再生機能
- シーン管理機能（Commonシーンをベースに他のシーンを使用する）
  - フェードイン/アウトによる画面遷移演出
- オプション機能
  - BGM音量
  - SE音量
  - タイトルに戻る
- オンラインマッチング機能（UGS Multiplayer Services）
  - クイックマッチ（空きルームへ自動参加 or 作成して待機）
  - ルーム一覧から手動参加
  - 20秒タイムアウト＋リトライ確認
- カードゲーム盤面（Main シーン）
  - カードテンプレート（UXML）+ カードデータベース（ScriptableObject）、カードにイラスト（Sprite）設定可能
  - 盤面背景画像（Addressables）
  - 手札：初期5枚をシャッフル後に扇状表示（画面下端・60%スケール）、ホバーで1.5倍拡大＋最前面表示、ドロップ後に残りカードが詰まるアニメーション
  - 相手手札：画面上部に裏向き・反転表示（非インタラクティブ）
  - フィールド：自分フィールド・相手フィールドの2エリア、各最大5枚を中央寄せ表示、ドラッグ&ドロップで手札から配置
  - デッキ：自分（右下）・相手（左上・反転）それぞれ裏向きで積み重ね表示
  - カード裏返しアニメーション（DOTween）
  - ドロー演出：デッキ位置から手札へ飛翔→フリップ（自分・相手ともに初期5枚を0.12秒ずつずらして並走。相手は裏向きのまま）
  - カード情報表示：カード名・コストバッジ（HeartIcon）・攻撃力バッジ（AttackIcon）・防御力バッジ（ShieldIcon）を縦一列左寄せで表示（各バッジ 36×36px に統一）
  - デッキ残り枚数：HeartIcon に枚数を重ねたバッジをデッキ上方に表示
  - 墓地ゾーン：プレイヤー墓地（左下）・相手墓地（右上）を茶色枠・暗い背景で常時表示。フィールドカードが破壊されると表向きで積み重ねて表示
  - 攻撃矢印：フィールドカードをドラッグするとベジェ曲線の矢印を表示。矢印は同時に1本のみ（別カードをドラッグすると差し替え）。Resolve まで表示し続ける
  - OK/戻るボタン：カード配置・攻撃操作をステージングし、OK で処理フェーズへ進む。戻るでカードを手札に飛翔アニメーションで戻す
  - ゲームロジック：1アクション→処理フェーズ（Ready→Resolve）→ターン交代。フィールド攻撃（ATK>=DEF で破壊）・デッキ攻撃（ATK枚分削減）・勝利条件（デッキ0枚）
  - Resolve テキスト演出：処理フェーズ開始時に画面中央に「Resolve」を黒文字＋白縁取りで表示
  - 攻撃アニメーション：アタッカーが目標方向へ突撃して戻るモーション（DOTween）
  - CPU 対戦：CPU エージェントが自動行動（フィールド攻撃→デッキ攻撃→カードプレイの優先順位）
