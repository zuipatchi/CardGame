# プロダクトドキュメント

## ゲーム概要

Unity 6 で開発するオンラインマルチプレイヤーカードゲーム。
UGS Multiplayer Services を使ったオンラインマッチメイキングに対応。

## 実装済み機能

- BGM / SE 再生（音量調整・永続化）
- Commonシーンをベースとしたアディティブシーン管理
- フェードイン/アウトによる画面遷移演出
- オプションモーダル（BGM音量・SE音量・タイトルに戻る）
- オンラインマッチング（クイックマッチ・ルーム一覧から手動参加・20秒タイムアウト）
- カードゲームのゲームプレイロジック
  - ターン管理・処理フェーズ（Ready/Resolve）
  - フィールド攻撃（ATK≥DEF で撃破）・デッキ攻撃（ATK 枚削減）・勝利条件（デッキ 0 枚）
  - CPU エージェント（フィールド攻撃→デッキ攻撃→カードプレイの優先順位で自動行動）
  - Resolve テキスト演出（処理フェーズ開始時に画面中央に表示）
  - 攻撃アニメーション（アタッカーが目標方向 60% へ突撃して戻るモーション）
  - 攻撃矢印（プレイヤー・CPU ともに Resolve まで表示し続ける）
